# python-fundamentals
Python basics practice — Data Engineering prep

# A variable is a named container that stores a value in memory so you can reuse or change it later 
— e.g. salary = 50000.

# A data type defines what kind of value a variable holds and what operations you can do with it 
— e.g. int (50000), float (50000.50), str ("Amit"), bool (True/False).

# An f-string is a way to insert variables directly into a string by putting f before the quotes and wrapping variable names in {}.
